# docs.olivetin.app

This is published at [docs.olivetin.app](https://docs.olivetin.app), built with Antora.

Current documentation sources are pulled from the [OliveTin](https://github.com/OliveTin/OliveTin) repository (`docs/` on branch `next`). Legacy 2.x documentation is still loaded from this repository’s `2k` branch until that line is fully migrated. This repository holds the Antora playbook, UI supplemental files, static assets, and the GitHub Pages publishing workflow.

## Issues and feedback

**Please open new issues in the [OliveTin](https://github.com/OliveTin/OliveTin) repository**, including reports about documentation content, typos, or gaps. The AsciiDoc sources for the published site live there under `docs/`, so fixes and discussion belong on that tracker. Use this **docs.olivetin.app** repository only for problems that are specific to the Antora playbook, site theme, GitHub Pages build, or other publishing infrastructure here.
