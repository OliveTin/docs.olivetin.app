# docs.olivetin.app

This is published at [docs.olivetin.app](https://docs.olivetin.app), built with Antora.

Current documentation sources are pulled from the [OliveTin](https://github.com/OliveTin/OliveTin) repository (`docs/` on branch `next`). Legacy 2.x documentation is still loaded from this repository’s `2k` branch until that line is fully migrated. This repository holds the Antora playbook, UI supplemental files, static assets, and the GitHub Pages publishing workflow.
